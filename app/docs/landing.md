# Welcome to Glyph

Glyph is a **lightweight documentation framework** that helps you create beautiful, fast, and modern documentation sites.

## Features

- **Fast** - Built with Vite for instant loading
- **Beautiful** - Clean design with Geist typography
- **Search** - Built-in offline search with Pagefind
- **Dark Mode** - Automatic light/dark theme
- **Components** - Cards, callouts, code blocks, and more

## Quick Start

```bash
npm create glyph-docs@latest my-docs
cd my-docs
npm run dev
```
